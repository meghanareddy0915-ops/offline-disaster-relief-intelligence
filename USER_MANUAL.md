# User Manual

## Starting the Application

Install dependencies:

```bash
pip install -r backend/requirements.txt
```

Start Streamlit:

```bash
streamlit run backend/app.py
```

---

## Uploading Reports

1. Click **Upload File**
2. Select a CSV, TXT or JSON file.
3. Wait for processing.

---

## Manual Input

1. Enter disaster information.
2. Click Analyze (if available).

---

## Supported Files

- CSV
- TXT
- JSON

---

## Output

The application displays processed disaster information.
